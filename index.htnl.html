Integracja z SAP Wariant 1: Użycie SAP Fiori SAP Fiori Elements: Jeśli używasz SAP Fiori, możesz stworzyć niestandardowy element Fiori i osadzić w nim ten kod. Użyj SAP Web IDE do stworzenia nowego projektu, dodaj pliki HTML, CSS i JavaScript do projektu i skonfiguruj ich użycie. SAPUI5: SAPUI5 pozwala tworzyć niestandardowe elementy interfejsu za pomocą standardowego HTML, CSS i JavaScript. Możesz stworzyć nowe widok (View) i kontroler (Controller) do obsługi logiki aplikacji. Wariant 2: Użycie Web Dynpro ABAP HTML Container: W Web Dynpro ABAP można użyć HTML Container do osadzenia kodu HTML. Stwórz nowy element UI HTMLContainer i wstaw do niego kod HTML. Logika ABAP: Logika może być zaimplementowana w ABAP i powiązana z twoim kodem HTML. Wariant 3: Użycie BSP (Business Server Pages) Aplikacja BSP: Stwórz nową aplikację BSP i dodaj pliki HTML, CSS i JavaScript do projektu. Integracja: Wstaw kod HTML na odpowiednią stronę BSP. Przykład integracji w BSP Tworzenie nowej aplikacji BSP: W SAP GUI, przejdź do transakcji SE80. Stwórz nową aplikację BSP. Dodanie HTML, CSS i JavaScript: Stwórz trzy pliki w swojej aplikacji BSP: index.html, styles.css, scripts.js. Skopiuj odpowiednie kody do tych plików. Osadzenie kodu HTML: Otwórz index.html i wstaw kod HTML. Dodaj odnośniki do plików CSS i JavaScript.link rel="stylesheet" href="styles.css"> script src="scripts.js">
